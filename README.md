Summary of the project - 
We are using the project to login our app with google
Plugins used - Socialite 

Steps to follow
1. change your app url in the env file 
2. add your database name 
3. Go to google developer console - create a project - fill all the required details and register it - in credentials you will create your google client and secret keys and then add those key in your env 

Thats it youre done !
